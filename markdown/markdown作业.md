# 当幸福来敲门读后感
1. 带有“励志”意味的电影，往往自带一种“鼓舞人”的力量。
2. 而在电影主人公实现自己的梦想是，电影的情感脉络也冲向高潮，观众也能在这样一个时刻收获一份“燃”的感觉。
>《当幸福来敲门》就是这样的电影
